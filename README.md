# ⚡ 简介
一个免费开源的react hooks，基于 React18、Hooks、TypeScript、 和 Vite 等主流技术

# 🌱 特性
- React：采用 React18 API
- Hooks：新的写法
- Vite：构建速度No1
- TypeScript：JavaScript 语言的超集
- PNPM：更快速的，节省磁盘空间的包管理工具
- ESlint：代码校验
- Prettier：代码格式化

# 🍰 功能
功能实现清单
- useToggle
- useBoolean

# 🚀 开发
```shell
# 配置
1. 一键安装 .vscode 目录中推荐的插件
2. node 版本 16+
3. pnpm 版本 8.x

# 克隆项目
git clone https://github.com/chuxin-cs/react-hooks

# 进入项目目录
cd react-hooks

# 安装依赖
pnpm i

# 启动服务
pnpm dev
```
